Tentang Saya

Awal mula

Saya adalah seorang mahasiswa jurusan Teknik Informatika di Universitas Gunadarma, saat ini sedang menjalani perkuliahan semester 6. Awal saya mulai tertarik pada Web Development adalah pada saat menginjak semester 5, dimana terdapat mata kuliah pemrograman web. Di awal saya belajar HTML & CSS, lalu setelah itu lanjut ke bahasa pemrograman PHP dan membuat sistem login & register. Dari situ lah mulai muncul ketertarikan dengan Web Development, akan tetapi disitu masih bingung dengan adanya istilah Fullstack, Frontend dan Backend.

Banyak yang bilang kita harus fokus di satu bidang saja, tapi bagaimana cara saya memilih bidang mana yang cocok? Maka dari itu yang saya lakukan pertama kali adalah belajar semuanya, baik itu Frontend maupun Backend. Dan setelah berjalannya waktu, mencoba belajar bahasa ini dan itu, mencoba framework abc, saya mendapat kesulitan di Backend, rasanya tidak mudah bagi saya untuk menjadi seorang Backend Developer. Saya lebih menyukai teknologi atau metode seperti apa yang akan ditampilkan pada website, dan bagaimana seseorang bisa nyaman saat berada di halaman website. Maka dari itu fokus saya sekarang adalah untuk menjadi seorang Frontend Developer.

Akan tetapi, untuk menjadi seorang Frontend Developer pun tidak mudah. Tidak hanya HTML & CSS saja, banyak teknologi yang harus dipelajari. Maka dari itu untuk sekarang saya sebisa mungkin mengikuti pelatihan gratis yang ada di internet, bahkan saya juga mengikuti pelatihan berbayar untuk mendapat sertifikat. Untuk sertifikat apa yang sudah saya miliki bisa dilihat di LinkedIn saya dan juga ada beberapa project portofolio saya yang bisa kalian lihat di bawah.
